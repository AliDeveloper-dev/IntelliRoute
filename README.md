## 🌟 Overview

**IntelliRoute** is a modern, intelligent **navigation and route optimization system** built using:

- **Flask** for backend APIs  
- **Leaflet.js** for interactive map visualization  
- **Advanced Graph Algorithms (DSA)** for route calculation  
- **Clean, smooth, professional UI**

It provides:

✔️ Shortest route selection  
✔️ Turn-by-turn directions  
✔️ Multiple vehicle types  
✔️ Real-time map visualizations

## 🏗️ Architecture
```
graph TD
    A[User Interface<br>(Leaflet.js, JS, HTML)] --> B[Flask Backend API]
    B --> C[Routing Engine<br>Dijkstra / A*]
    C --> D[Graph Data Layer]
    B --> E[Vehicle Logic]
    A --> F[Visualization Renderer]
    F --> A

```
